Project Motivations

I'm building a distraction-free, easy to use, and focused music experience! I've recently become interested in 'The Unix Philosophy,' the concepts and ideas that informed the early development of the Unix operating system. It was sumarized by computer scientist and technology historian Peter H. Salus[1] as:

  • Write programs that do one thing and do it well
  • Write programs to work together
  • Write programs that handle text streams, because that is a universal interface

I believe these concepts, particularly the first point, can apply not only to software development, but also to the tools we use every day as a whole. For example, a dedicated screwdriver is better than the one in a Swiss Army knife.

My phone has become a Swiss Army knife, and I want to replace it. I've started carrying a notepad instead of using the notes app. I use an old digital camera to take pictures when possible instead of my phone camera. And now, I want to use my own digital audio player to replace the music app.

Work in progress, more details coming soon
